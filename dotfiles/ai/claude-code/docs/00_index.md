# Documentation Index

> Reference documentation for all development patterns, conventions, and guides.

## Structure

```
docs/
├── reference/     Canonical technical references
├── architecture/  Architecture decision records
├── patterns/      Software patterns (general + per-language)
├── conventions/   Coding standards (general + per-language)
├── testing/       Testing strategies (general + per-language)
└── guides/        Step-by-step operational guides
```

Each of `patterns/`, `conventions/`, and `testing/` contains:
- `general/` — language-agnostic content
- Per-language folders: `go/`, `python/`, `typescript/`, `rust/`, `elixir/`, `nodejs/`, `nix/`, `pkl/`

Each folder has a `00_index.md` as its main entry point.
